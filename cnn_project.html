SQL Database development

The .csv files were used to populate two tables into a Database created with MySQL. A new table was created using JOIN query between the two initial tables. The main table was contain only the information adults and it is the table used during the process of data investigation.


A data cleaning and data transformation was used while creating the main table. For variables that concataning both values of "refused to answear" and "Did not know", they were transformed into a single variable. The income variable was substitued from a range value to an average value. The main KPI for this data was also define here, the sum of all the mental health questions is the most relevant variable for the data exploration, as it is directly related to the different mental helath groups defined.
