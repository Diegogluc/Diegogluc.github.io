The NHANES Dataset

This project uses a curated version of the dataset from the National Health and Nutrition Examination Surveys (NHANES) survey. The NHANES is a long-term program conducted by the Centers for Disease Control and Prevention (CDC). It collects data about the health of adults and children in the United States, combining interviews, physical examinations, and laboratory tests to assess health and nutritional outcomes.

This curated version was provided by the Tera platform for online courses and contains two .csv files. The first file contains mental health questions data only from adults (5,334 data points). The second .csv file contains eating and exercise habits data of both adults and children (9,424 data points). The original dataset can be accessed using this link.


SQL Database development

The .csv files were used to populate two tables into a Database created with MySQL. A new table was created using JOIN query between the two initial tables. The main table contained only information about adults and and was used during the data investigation proces.


A data cleaning and data transformation was used while creating the main table. For variables that concataning both values of "refused to answear" and "Did not know", they were transformed into a single variable. The income variable was substitued from a range value to an average value. The main KPI for this data was also defined here, the sum of all the mental health questions were directly related to groups of mental helath symptoms.

  • No symptoms
  • Mild symptoms
  • Moderate to Severe symptoms

Data Investigation

Exploratory Data Analysis

This first step was an evaluation of each variable of the dataset to understand trends, distributions and spot anomalies. Understanding the distribution of both cathegorical and numerical variables was crucial to define the most relevant attributes.

After this analysis, the impact between variables and the KPI relating mental issues was narrowed down to a small list:

  • Weekly hours of exercise
  • Gender
  • Ethnicity
  • Age
  • Education level

Bivariant Analysis

This phase focused on understanding the relationship between featured variable listed and the three groups of symptoms.

The visual inspection of a possible correlation in the bivariant analysis will feed the statistical analysis used latter. For the numerical variables a boxplot was used to check min, max values as well as 25, 50 and 75 percentiles. For the cathegorical variables bar charts were used to check diferences in the distributions of the cathegories.

Statistical Tests

ANOVA tests were used compare the mean value of each variable between the three different groups of depression symptoms. If the null hypothesis was rejected (p value < significance level) a pairwise comparisons with TukeyHSD was used to define wich groups have different mean values.

For the cathegorical variables a Chi-square test was performed to check if the depression groups changed in regard the categorical variable tested.
